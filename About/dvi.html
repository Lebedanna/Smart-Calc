Smart Calc v2.0

О приложении

Приложение разработано на языке C++ стандарта C++17 с использованием паттерна MVC (model, view, controller). Графический интерфейс реализован с помощью библиотеки QT.

    Реализованный функционал:
  • Ввод целых и вещественных чисел, экспоненциальная форма записи числа;
  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации;
  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации с подстановкой значения переменной Х в виде числа;
  • Построение графика функции, заданной с помощью выражения в инфиксной нотации с переменной Х (с координатными осями, отметкой используемого масштаба и сеткой с адаптивным шагом);
  • Ввод с клавиатуры;
  • Возможность ввода до 255 символов.

Операторы

  • Скобки ()
  • Сложение +
  • Вычитание -
  • Умножение *
  • Деление /
  • Возведение в степень ^
  • Остаток от деления %
  • Унарный плюс +a
  • Унарный минус -a

Функции

  • cos
  • sin
  • tan
  • acos
  • asin
  • atan
  • sqrt
  • ln
  • log

Доступные команды Make:

Установка

Удаление

make uninstall

Unit-тесты

GCOV-report о покрытии тестами

make gcov_report

Valgrind - проверка на утечки памяти

make valgrind

Проверка стиля кода
